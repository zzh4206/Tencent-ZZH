# 星火计划预习题 — 解答

本目录包含 2025 星火量子方向预习题目的完整解答。每题一个独立 Python 脚本，可直接运行。

## 目录结构

```
├── README.md
├── 1_环境配置/
│   └── solution_1.py          # TensorCircuit + TF 环境验证
├── 2_向量和矩阵/
│   ├── 2-1_旋转矩阵.py         # R(θ)v 几何意义（顺时针旋转）
│   ├── 2-2_矩阵指数.py         # e^{iθP} = cosθ·I + i sinθ·P
│   ├── 2-3_矩阵期望.py         # ⟨0|e^{-iθ/2 P} Q e^{iθ/2 P}|0⟩
│   ├── 2-4_张量积.py          # Ising Hamiltonian 期望值
│   ├── 2-5_狄拉克符号.py      # Dirac 符号改写 + (|010⟩-|101⟩)/√2
│   └── 2-6_TensorCircuit重写.py # 用 TC 后端重写 §2
├── 3_导数与梯度下降/
│   ├── 3-1_数值微分.py         # 前向/中心差分数值求导
│   ├── 3-2_三角函数数值微分.py  # 参数平移法则 (δ=π/2, τ=2)
│   ├── 3-3_单比特参数平移.py    # 9 种 P₁/P₂ 组合的 f(θ) 分析
│   └── 3-4_梯度下降.py         # GD 求 f(θ) 最小值
├── 4_测量/
│   ├── 4-1_线路期望计算.py     # Bell 态 ⟨Z₀Z₁⟩ = 1
│   ├── 4-2_测量估计期望.py     # 采样估计期望值
│   └── 4-3_测量不确定度.py     # 1/√N 标度律 (SQL)
├── OPT-1_Grover搜索/
│   ├── OPT-1-1_oracle.py       # 3×2 格子染色 Oracle
│   └── OPT-1-2_Grover算法.py   # Grover 搜索实现

    ├── OPT-2-1_求解方程.py     # ax³+bx²+cx+d=0 变分解
    ├── OPT-2-2_优化点.py       # 几何中位数（挖井）
    └── OPT-2-3_优化线.py       # L1 最优直线（公路）
```

## 环境要求

```bash
pip install numpy matplotlib tensorcircuit tensorflow scipy cotengra
```

## 运行方式

每道题的脚本可独立运行：
```bash
python "2_向量和矩阵/2-1_旋转矩阵.py"
```

## 关键结果速查

| 题目 | 核心结论 |
|------|---------|
| 2-1 | R(θ) 顺时针旋转矩阵 |
| 2-2 | e^{iθP}=cosθ·I+i sinθ·P，成立条件 P²=I |
| 2-3 | Bloch 球旋转：P=Q 时期望常数，P≠Q 时以 2π 周期振荡 |
| 2-4 | ⟨0...0\|H\|0...0⟩ = n（Ising 模型在 |0⟩ 态的能量） |
| 2-5 | (|010⟩-|101⟩)/√2 = (0,0,1/√2,0,0,0,-1/√2,0)^T |
| 3-1 | 中心差分 O(δ²)，δ 最优 ~1e-6 |
| 3-2 | 参数平移: δ=π/2, τ=2 → 精确导数 |
| 3-3 | f(θ) 均为 A sinθ+B cosθ+C 形式，参数平移精确成立 |
| 3-4 | GD 成功找最小值，η=0.1~0.5 收敛良好 |
| 4-1 | Bell 态：⟨Z₀Z₁⟩ = 1 |
| 4-2 | 采样估计期望值 |
| 4-3 | σ ∝ 1/√N（标准量子极限） |
| OPT-1 | Grover k_opt=4，成功概率 ~1 |
| OPT-2 | 变分优化求解方程/几何中位数/L1直线 |
